Ink + text

--text-hi
Primary heading-class text at full ink contrast.
--text
Body / paragraph text. Most ordinary content uses this tier.
--text-mid
Metadata, secondary descriptions, captions.
--text-lo
Faint elements — separators, dot indicators, low-emphasis labels.

Brand accent (unified across modes)

accent-warm accent-warm-soft

Inline code text uses --accent-warm. Hover stripes on list rows use the same token.

Semantic (unified across modes)

accent-cool accent-danger

Section labels and status pills (the SCHOLAR / FORTHCOMING badges on the publications page) use --accent-cool. --accent-danger is reserved for future destructive UI.
